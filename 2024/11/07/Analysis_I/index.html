Part 1 数列极限

定义1.1.1 （数列极限）

设 \(\{ a_{n} \}\subset \mathbf{R}\) ， \(a\in \mathbf{R}\) 。若 \(\forall \epsilon>0\) ，\(\exists N\) ，使得对于 \(\forall n>N\) 都有 \[ |a_{n}-a|<\epsilon \] 则称在 \(n\) 趋向于无穷大时，\(\{ a_{n} \}\) 以 \(a\) 为极限，记为 \[ \lim_{ n \to \infty } a_{n}=a,a_{n}\to a(n\to +\infty) \] 存在极限的数列，称为收敛数列，反之为发散数列。

定义1.1.2 （发散数列的定义/1.1.1的否定）

设 \(\{ a_{n} \} \subset \mathbf{R}\) 。若对于 \(a\in \mathbf{R}\) ， \(\exists \epsilon_{0}>0\) ，使得对于 \(\forall N\) ， \(\exists n_{N}>N\) 满足 \[ |a_{n_{N}}-a|\ge \epsilon_{0}, \] 则称当 \(n\) 趋于无穷大时， \(\{ a_{n} \}\) 不收敛于 \(a\) ，\(a_{n}\nrightarrow l(n\to +\infty)\)。

性质1.1.3 （收敛数列的性质）

若 \(a_{n}\to a\) ， \(\{ a_{n} \}\) 的任何子列都收敛于 \(a\) 。

性质1.1.4 （1.1.3的否定）

若 \(\{ a_{n} \}\) 存在两个子列收敛到不同极限，则 \(\{ a_{n} \}\) 发散。

性质1.1.5 （极限的唯一性）

一个收敛数列的极限是唯一的。

性质1.1.6（极限的四则运算）

若 \(\{ a_{n} \},\{ b_{n} \}\) 收敛，且 \(a_{n}\to a,b_{n}\to b\) ，那么： \[ \begin{aligned} &\lim_{ n \to \infty } (a_{n} \pm b_{n})=a+b \\ &\lim_{ n \to \infty } (a_{n} b_{n})=ab \\ &\lim_{ n \to \infty } \frac{a_{n}}{b_{n}}=\frac{a}{b},(b \ne 0) \end{aligned} \]

性质1.1.7 （收敛数列的不等式性质）

若 \(\{ a_{n} \},\{ b_{n} \}\) 收敛，且当 \(n\) 充分大时，有 \(a_{n}\le b_{n}\) ，则 \(a\le b\) 。（保号性）

若 \(\{ a_{n} \},\{ b_{n} \}\) 收敛，且当 \(n\) 充分大时，有 \(a_{n}< b_{n}\) ，则 \(a\le b\) 。（模糊保号性）

性质1.1.8 （夹逼定理）

若对于充分大的 \(n\) 有 \(a_{n}\le b_{n}\le c_{n}\) ，且 \[ \lim_{ n \to \infty } a_{n}=\lim_{ n \to \infty } c_{n}=x \] 则 \[ \lim_{ n \to \infty } b_{n}=x \]

定义1.2.1 （有界数列）

\(\{ a_{n} \}\subset \mathbf{R}\) ，若 \(\exists A\in R\) ，使得 \(a_{n}\le A\) 对于 \(\forall n \in \mathbf{N^{*}}\) 成立，则称 \(A\) 是 \(\{ a_{n} \}\) 的一个上界。下界同理。 若 \(\{ a_{n} \}\) 既有上界又有下界，则称其有界。

性质1.2.2 （收敛必有界）

收敛数列必有界。

定义1.2.3 （确界）

若 \(E \ne \emptyset\) ，若 \(\alpha\in \mathbf{R}\) 满足：

（1）对于 \(\forall x \in E\) 都有 \(x\le \alpha\)

（2）对于 \(\forall \epsilon>0\) ， \(\exists x_{\epsilon}\in E\) ，使得 \(x_{\epsilon}>\alpha-\epsilon\) 则称 \(\alpha\) 为 \(E\) 的上确界， \(\alpha=\sup E\) 。

下确界同理。

（事实上，（1）条件使得 \(\sup E\ge \alpha\) （2）条件使得 \(\sup E \le \alpha\) ）

性质1.2.4 （确界的性质）

\[ \begin{aligned} \inf\{ a_{n} \}&\le a_{n}\\ \sup\{ a_{n} \}&\ge a_{n}\\ \inf\{ a_{n} +x\}&=\inf\{ a_{n} \}+x,\text{x is constant} \\ \sup\{ a_{n} +x\}&=\sup\{ a_{n} \}+x,\text{x is constant}\\ \inf\{ a_{n}+b_{n} \}&\ge \inf \{ a_{n} \}+\inf \{ b_{n} \} \\ \sup\{ a_{n}+b_{n} \}&\le \sup \{ a_{n} \}+\sup \{ b_{n} \} \\ \end{aligned} \] （性质与 \(\limsup\) ，\(\liminf\) 的性质极为相似，更多不等式见上下极限）

性质1.2.5 （确界的不等式性质）

如果 \(a_{n} > b_{n}\) ，则 \(\inf\{ a_{n} \}\ge\inf\{ b_{n} \}\) ， \(\sup\{ a_{n} \}\ge\sup\{ b_{n} \}\) 。（同样存在模糊保号性）

如果 \(a_{n} < b_{n}\) ，则 \(\inf\{ a_{n} \}\le\inf\{ b_{n} \}\) ， \(\sup\{ a_{n} \}\le\sup\{ b_{n} \}\) 。

性质1.2.6 （单调数列的确界与极限）

若 \(\{ a_{n} \}\) 单调递增且收敛，则 \(a_{n}\to \sup\{ a_{n} \}\)

若 \(\{ a_{n} \}\) 单调递减且收敛，则 \(a_{n}\to \inf\{ a_{n} \}\)

定义1.2.7 （基本列/Cauchy列）

\(\{ a_{n} \}\subset \mathbf{R}\) 。若 \(\forall \epsilon>0\) ， \(\exists N\) 使得 \(\forall m,n>N\) 都有 \[ |a_{m}-a_{n}|<\epsilon \] 则称 \(\{ a_{n} \}\) 是基本列/Cauchy列。

性质1.2.8 （1.2.7的否定）

\(\{ a_{n} \}\subset \mathbf{R}\) 。若 \(\exists \epsilon_{0}>0\) ，使得对于 \(\forall N\) ， \(\exists m_{N},n_{N}>N\) 都有 \[ |a_{m}-a_{n}|\ge \epsilon_{0} \] 则称 \(\{ a_{n} \}\) 不是基本列/Cauchy列。

定理1.2.9 （Cauchy收敛准则）

数列收敛的充要条件是该数列为基本列。

定理1.3.1 （单调有界定理）

若 \(\{ a_{n} \}\) 单调递增（减），则 \(\{ a_{n} \}\) 存在极限的充要条件是 \(\{ a_{n} \}\) 有上（下）界。

定理1.3.2 （闭区间套定理）

若 \([a_{n+1},b_{n+1}]\subset[a_{n},b_{n}]\) ，且 \(b_{n}-a_{n}\to 0\) ，则存在唯一的 \(\xi \in[a_{n},b_{n}], \forall n\) ，且 \(a_{n},b_{n}\to \xi\) 。

如果没有 \(b_{n}-a_{n}\to 0\) 的条件，则闭区间交为闭区间。

定理1.3.3 （确界存在定理）

有上（下）界的集合必存在上（下）确界。

定理1.3.4 （Bolzano-Weierstrass/列紧性定理/凝聚定理）

有界数列必包含收敛子列。（广义：一个数列必包含广义收敛子列）

定理1.3.5 （Heine-Borel有限覆盖定理）

有限闭区间的任意开覆盖中，必定存在有限个子覆盖。

（Cauchy收敛准则在前文）

定理1.3.6 （Lesbege数/Heine-Borel定理加强）

\([a,b]\) 的一个开覆盖 \(\{ I_{n} \}\) 中，必存在 \(\sigma>0\) ，使得只要 \(A\subset[a,b]\) ，且 \(|A|<\sigma\) ，必有 \(\{ I_{n} \}\) 中的一个开区间包含 \(A\) 。其中 \(\sigma\) 称为 Lebesgue 数。

定义1.4.1 （广义极限）

\(\{ a_{n} \} \subset \mathbf{R}\) ，若 \(\forall A>0\) ，\(\exists N\) 使得 \(\forall n>N\) 都有 \(a_{n}>A\) ，则称 \(a_{n}\to +\infty\) ，有广义极限 \(+\infty\) 。

\(\{ a_{n} \} \subset \mathbf{R}\) ，若 \(\forall A<0\) ，\(\exists N\) 使得 \(\forall n>N\) 都有 \(a_{n}<A\) ，则称 \(a_{n}\to -\infty\) ，有广义极限 \(-\infty\) 。

\(\{ a_{n} \} \subset \mathbf{R}\) ，若 \(\forall A>0\) ，\(\exists N\) 使得 \(\forall n>N\) 都有 \(|a_{n}|>A\) ，则称 \(a_{n}\to \infty\) 。

\(\mathbf{R_{\infty}}:=\mathbf{R}\cup\{+\infty,-\infty \}\)

同理定义 \(\inf a_{n}=\infty\) ，\(\sup a_{n}=\infty\) 。

定义1.4.2 （无穷小数列的定义）

若 \(a_{n}\to 0\) ，则称 \(\{ a_{n} \}\) 是无穷小数列，简称无穷小。

定义1.4.3 （无穷大数列的定义）

若 \(a_{n}\to \infty\) ，则称 \(\{ a_{n} \}\) 是无穷大数列，简称无穷大。

性质1.4.5 （无穷大/无穷小的性质）

（1）无穷小（大）的和（差/积）为无穷小（大）

（2）\(\{ a_{n} \}\) 无穷小（大）， \(\{ c_{n} \}\) 有界， \(\{ a_{n}c_{n} \}\) 为无穷小（大）

（3）若对于充分大的 \(n\) ，有 \(0\le |a_{n}|\le |b_{n}|\) 。若 \(\{ b_{n} \}\) 无穷小，则 \(\{ a_{n} \}\) 为无穷小；若 \(\{ a_{n} \}\) 无穷大，则 \(\{ b_{n} \}\) 无穷大。

（4）无穷大数列必无界；无穷大数列任意子列必为无穷大数列；无界数列必有无穷大子列。

定义1.5.1 （极限点集合 \(E\) ）

\[ E:=\{ l\in \mathbf{R_{\infty}} | \exists \{ a_{k_{n}} \},\lim_{ n \to \infty } a_{k_{n}} =l \} \]

性质1.5.2 （ \(E\) 的性质）

\(E\) 非空。 \(\sup E \in E,\inf E\in E\) 。

定义 1.5.3 （上下极限定义一）

\[ \begin{aligned} \limsup_{ n \to \infty } a_{n} &:= \sup E \\ \liminf_{ n \to \infty } a_{n} &:= \inf E \end{aligned} \]

定义 1.5.4 （上下极限定义二）

\[ \begin{aligned} &\limsup_{ n \to \infty } a_{n} :=\lim_{ n \to \infty } \sup_{k\ge n} a_{k} = \inf_{ n } \sup_{k\ge n} a_{k} \\ &\liminf_{ n \to \infty } a_{n} :=\lim_{ n \to \infty } \inf_{k\ge n} a_{k} = \sup_{ n } \inf_{k\ge n} a_{k} \\ \end{aligned} \]

定义 1.5.5 （上下极限定义三）

若 \(l\) 满足：

（1）对于 \(\forall \epsilon>0\) ， \(\exists N\) ，使得 \(\forall n>N\) 有 \[ a_{n}<l+\epsilon \] （2）对于 \(\forall \epsilon>0\) ，\(\exists \{ a_{k_{n}} \}\) 使得 \[ a_{k_{n}} > l-\epsilon \] 则 \[ \limsup_{ n \to \infty } a_{n} := l \] 无穷形式类似极限的定义。

下极限同理。

性质1.5.6 （上下极限与极限）

\[ \liminf_{ n \to \infty } a_{n} \le \limsup_{ n \to \infty } a_{n} \] 取等条件条件：当且仅当， \(\{ a_{n} \}\) 收敛，其极限等于其上下极限的值。

性质1.5.7 （上下极限的性质）

\[ \begin{aligned} \liminf_{ n \to \infty } a_{n} + \limsup_{ n \to \infty } b_{n} &\le\limsup_{ n \to \infty } (a_{n}+b_{n}) \\ &\le\limsup_{ n \to \infty } a_{n}+\limsup_{ n \to \infty } b_{n}\\ \liminf_{ n \to \infty } a_{n} + \limsup_{ n \to \infty } b_{n} &\ge\liminf_{ n \to \infty } (a_{n}+b_{n}) \\ &\ge\liminf_{ n \to \infty } a_{n}+\liminf_{ n \to \infty } b_{n} \end{aligned} \]

性质1.5.8 （上下极限的不等式性质）

若对于充分大的 \(n\) 有 \[ a_{n}< b_{n} \] 则 \[ \begin{aligned} &\limsup_{ n \to \infty } a_{n} \le \limsup_{ n \to \infty } b_{n} \\ &\liminf_{ n \to \infty } a_{n} \le \liminf_{ n \to \infty } b_{n} \end{aligned} \] （同样具有模糊保号性）

定理1.6.1 （Cauchy命题）

如果 \(\{ a_{n} \}\) 收敛，则其均值极限存在且等于： \[ \lim_{ n \to \infty } \frac{a_{1}+\dots +a_{n}}{n}=\lim_{ n \to \infty } a_{n} \]

如果 \(\{ a_{n} \}\) 收敛，且 \(a_{n}>0\)，则其均值极限存在且等于： \[ \lim_{ n \to \infty } \sqrt[n]{ a_{1}\dots a_{n} }=\lim_{ n \to \infty } a_{n} \]

定理1.6.2 （Toeplitz定理）

设 \(n,k\in \mathbf{N^{*}}\) ，\(t_{n,k}\ge 0\) ，且 \(\sum_{k=1}^{n} t_{n,k}=1\) ，\(\lim_{ n \to \infty } t_{n,k}=0\) 如果 \(\{ a_{n} \}\) 收敛，则 \[ \lim_{ n \to \infty } \sum_{k=1}^{n} t_{n,k}a_{k}=\lim_{ n \to \infty } a_{n} \]

定理1.6.3 （Stolz定理 \(\frac{*}{\infty}\) 型）

如果 \(\{ b_{n} \}\) 严格递增且 \(b_{n}\to +\infty\) ，且 \(\frac{a_{n+1}-a_{n}}{b_{n+1}-b_{n}}\) 收敛，则 \[ \lim_{ n \to \infty } \frac{a_{n}}{b_{n}} = \lim_{ n \to \infty } \frac{a_{n+1}-a_{n}}{b_{n+1}-b_{n}} \]

定理1.6.3 （Stolz定理 \(\frac{*}{0}\) 型）

如果 \(\{ b_{n} \}\) 严格递减且 \(b_{n}\to 0\) ，且 \(\frac{a_{n+1}-a_{n}}{b_{n+1}-b_{n}}\) 收敛，则 \[ \lim_{ n \to \infty } \frac{a_{n}}{b_{n}} = \lim_{ n \to \infty } \frac{a_{n+1}-a_{n}}{b_{n+1}-b_{n}} \]

定理1.6.4 （不动点原理）

若 \(\{ x_{n} \}\) 满足 \(x_{n+1}=f(x_{n})\) ， \(x_n\to\xi\) ，\(f(x_{n})\to f(\xi)\) ，则 \(\xi\) 一定满足： \[ f(\xi)=\xi \]

定理1.6.5 （压缩映射原理）

若 \(f([a,b])\subset[a,b]\) ，且存在一个 \(k\in(0,1)\) 满足 \(\forall x,y\in[a,b]\) ，\(|f(x)-f(y)|\le k|x-y|\) ，则称 \(f\) 在 \([a,b]\) 上是一个压缩映射， \(k\) 为压缩常数。

压缩映射函数 \(f\) 满足以下命题： （1） \(f\) 在 \([a,b]\) 存在唯一的不动点 \(\xi=f(\xi)\)

（2）若 \(\{ a_{n} \}\) 递推式为 \(a_{n+1}=f(a_{n})\) ，则 \(\{ a_{n} \}\) 收敛于 \(\xi\)

（3） \(|a_{n}-\xi|\le \frac{k}{1-k}|a_{n}-a_{n-1}|\) ， \(|a_{n}-\xi|\le \frac{k^{n}}{1-k}|a_{1}-a_{0}|\)

定义1.7.1 （ \(e\) 的定义）

考虑两个数列： \[ \begin{aligned} e_{n}&:=\left( 1+\frac{1}{n} \right)^{n}\\ s_{n}&:=1+\frac{1}{1!}+\frac{1}{2!}+\dots+\frac{1}{n!} \end{aligned} \] 有以下性质 （1） \(e_{n},s_{n}\) 均严格递增，有界，收敛

（2）\(e_{n}\le s_{n}\)

（3） \(e_{n},s_{n}\) 收敛且相等

定义： \[ e:=\lim_{ n \to +\infty } e_{n} =\lim_{ n \to +\infty } s_{n} \]

性质1.7.2 （ \(e\) 是无理数）

\(e\) 是无理数

性质1.7.3 （有关不等式）

\[ \begin{aligned} &\left( 1+\frac{1}{n} \right)^{n}<e<\left( 1+\frac{1}{n} \right)^{n+1} \\ &\frac{1}{n+1}<\ln \left( 1+\frac{1}{n} \right)< \frac{1}{n} \end{aligned} \]

性质1.7.4 （ \(e\) 的估计）

\[ e=1+\frac{1}{1!}+\frac{1}{2!}+\dots+\frac{1}{n!}+\frac{\theta_{n}}{n!n} \] 其中 \(\theta_{n} \in\left( \frac{n}{n+1} , 1\right)\)

定义1.7.5 （Euler常数）

\[ \gamma := \lim_{ n \to +\infty } \sum_{i=1}^{n} \frac{1}{n}-\ln n \] \(\gamma\) 称为 Euler-Mascheroni 常数（Euler常数）
