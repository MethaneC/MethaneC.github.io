Part 1 数列极限

定义1.1.1 （数列极限）

设 {an}⊂R\{ a_{n} \}\subset \mathbf{R}{an​}⊂R ， a∈Ra\in \mathbf{R}a∈R 。若 ∀ϵ>0\forall \epsilon>0∀ϵ>0 ，∃N\exists N∃N ，使得对于 ∀n>N\forall n>N∀n>N 都有

∣an−a∣<ϵ|a_{n}-a|<\epsilon ∣an​−a∣<ϵ

则称在 nnn 趋向于无穷大时，{an}\{ a_{n} \}{an​} 以 aaa 为极限，记为

lim⁡n→∞an=a,an→a(n→+∞)\lim_{ n \to \infty } a_{n}=a,a_{n}\to a(n\to +\infty) n→∞lim​an​=a,an​→a(n→+∞)

存在极限的数列，称为收敛数列，反之为发散数列。

定义1.1.2 （发散数列的定义/1.1.1的否定）

设 {an}⊂R\{ a_{n} \} \subset \mathbf{R}{an​}⊂R 。若对于 a∈Ra\in Ra∈R ， ∃ϵ0>0\exists \epsilon_{0}>0∃ϵ0​>0 ，使得对于 ∀N\forall N∀N ， ∃nN>N\exists n_{N}>N∃nN​>N 满足

∣anN−a∣≥ϵ0,an↛l(n→+∞)|a_{n_{N}}-a|\ge \epsilon_{0},a_{n}\nrightarrow l(n\to +\infty) ∣anN​​−a∣≥ϵ0​,an​↛l(n→+∞)

则称当 nnn 趋于无穷大时， {an}\{ a_{n} \}{an​} 不收敛于 aaa 。

性质1.1.3 （收敛数列的性质）

若 an→aa_{n}\to aan​→a ， {an}\{ a_{n} \}{an​} 的任何子列都收敛于 aaa 。

推论1.1.4 （1.1.3的否定）

若 {an}\{ a_{n} \}{an​} 存在两个子列收敛到不同极限，则 {an}\{ a_{n} \}{an​} 发散。

性质1.1.5 （极限的唯一性）

一个收敛数列的极限是唯一的。

性质1.1.6（极限的四则运算）

若 {an},{bn}\{ a_{n} \},\{ b_{n} \}{an​},{bn​} 收敛，且 an→a,bn→ba_{n}\to a,b_{n}\to ban​→a,bn​→b ，那么：

\begin{align} &\lim_{ n \to \infty } (a_{n} \pm b_{n})=a+b \\ &\lim_{ n \to \infty } (a_{n} b_{n})=ab \\ &\lim_{ n \to \infty } \frac{a_{n}}{b_{n}}=\frac{a}{b},(b \ne 0) \end{align}

性质1.1.7 （收敛数列的不等式性质）

若 {an},{bn}\{ a_{n} \},\{ b_{n} \}{an​},{bn​} 收敛，且当 nnn 充分大时，有 an≤bna_{n}\le b_{n}an​≤bn​ ，则 a≤ba\le ba≤b 。（保号性）
若 {an},{bn}\{ a_{n} \},\{ b_{n} \}{an​},{bn​} 收敛，且当 nnn 充分大时，有 an<bna_{n}< b_{n}an​<bn​ ，则 a≤ba\le ba≤b 。（模糊保号性）

性质1.1.8 （夹逼定理）

若对于充分大的 nnn 有 an≤bn≤cna_{n}\le b_{n}\le c_{n}an​≤bn​≤cn​ ，且

lim⁡n→∞an=lim⁡n→∞cn=x\lim_{ n \to \infty } a_{n}=\lim_{ n \to \infty } c_{n}=x n→∞lim​an​=n→∞lim​cn​=x

则

lim⁡n→∞bn=x\lim_{ n \to \infty } b_{n}=x n→∞lim​bn​=x

（可以证明极限存在，并求值）

定义1.2.1 （有界数列）

{an}⊂R\{ a_{n} \}\subset \mathbf{R}{an​}⊂R ，若 ∃A∈R\exists A\in R∃A∈R ，使得 an≤Aa_{n}\le Aan​≤A 对于 ∀n∈N∗\forall n \in \mathbf{N^{*}}∀n∈N∗ 成立，则称 AAA 是 {an}\{ a_{n} \}{an​} 的一个上界。下界同理。
若 {an}\{ a_{n} \}{an​} 既有上界又有下界，则称其有界。

性质1.2.2 （收敛必有界）

收敛数列必有界。

定义1.2.3 （确界）

若 E≠∅E \ne \emptysetE=∅ ，若 α∈R\alpha\in \mathbf{R}α∈R 满足：
（1）对于 ∀x∈E\forall x \in E∀x∈E 都有 x≤αx\le \alphax≤α
（2）对于 ∀ϵ>0\forall \epsilon>0∀ϵ>0 ， ∃xϵ∈E\exists x_{\epsilon}\in E∃xϵ​∈E ，使得 xϵ>α−ϵx_{\epsilon}>\alpha-\epsilonxϵ​>α−ϵ
则称 α\alphaα 为 EEE 的上确界， α=sup⁡E\alpha=\sup Eα=supE 。

下确界同理。

性质1.2.4 （确界的性质）

\begin{align} \inf\{ a_{n} \}&\le a_{n}\\ \sup\{ a_{n} \}&\ge a_{n}\\ \inf\{ a_{n} +x\}&=\inf\{ a_{n} \}+x,\text{x is constant} \\ \sup\{ a_{n} +x\}&=\sup\{ a_{n} \}+x,\text{x is constant}\\ \inf\{ a_{n}+b_{n} \}&\ge\inf\{ a_{n}+ \}\ge \inf a_{n}+\inf b_{n} \\ \sup\{ a_{n}+b_{n} \}&\le \sup a_{n}+\sup b_{n} \\ \end{align}

性质1.2.5 （确界的不等式性质）

如果 an>bna_{n} > b_{n}an​>bn​ ，则 inf⁡{an}≥inf⁡{bn}\inf\{ a_{n} \}\ge\inf\{ b_{n} \}inf{an​}≥inf{bn​} ， sup⁡{an}≥sup⁡{bn}\sup\{ a_{n} \}\ge\sup\{ b_{n} \}sup{an​}≥sup{bn​} 。（同样存在模糊保号性）
如果 an<bna_{n} < b_{n}an​<bn​ ，则 inf⁡{an}≤inf⁡{bn}\inf\{ a_{n} \}\le\inf\{ b_{n} \}inf{an​}≤inf{bn​} ， sup⁡{an}≤sup⁡{bn}\sup\{ a_{n} \}\le\sup\{ b_{n} \}sup{an​}≤sup{bn​} 。

性质1.2.6 （单调数列的确界与极限）

若 {an}\{ a_{n} \}{an​} 单调递增，则 an→sup⁡{an}a_{n}\to \sup\{ a_{n} \}an​→sup{an​}
若 {an}\{ a_{n} \}{an​} 单调递减，则 an→inf⁡{an}a_{n}\to \inf\{ a_{n} \}an​→inf{an​}

定义1.2.7 （基本列/Cauchy列）

{an}⊂R\{ a_{n} \}\subset \mathbf{R}{an​}⊂R 。若 ∀ϵ\forall \epsilon∀ϵ ， ∃N\exists N∃N 使得 ∀m,n>N\forall m,n>N∀m,n>N 都有

∣am−an∣<ϵ|a_{m}-a_{n}|<\epsilon ∣am​−an​∣<ϵ

则称 {an}\{ a_{n} \}{an​} 是基本列/Cauchy列。

性质1.2.8 （1.2.6的否定）

{an}⊂R\{ a_{n} \}\subset \mathbf{R}{an​}⊂R 。若 ∃ϵ0>0\exists \epsilon_{0}>0∃ϵ0​>0 ，使得对于 ∀N\forall N∀N ， ∃mN,nN>N\exists m_{N},n_{N}>N∃mN​,nN​>N 都有

∣am−an∣≥ϵ0|a_{m}-a_{n}|\ge \epsilon_{0} ∣am​−an​∣≥ϵ0​

则称 {an}\{ a_{n} \}{an​} 不是基本列/Cauchy列。

定理1.2.9 （Cauchy收敛准则）

数列收敛的充要条件是该数列为基本列。

定理1.3.1 （单调有界定理）

若 {an}\{ a_{n} \}{an​} 单调递增（减），则 {an}\{ a_{n} \}{an​} 存在极限的充要条件是 {an}\{ a_{n} \}{an​} 有上（下）界。

定理1.3.2 （确界存在定理）

有上（下）界的数列必存在上（下）确界。

定理1.3.3 （Bolzano-Weierstrass，列紧性定理）

有界数列必包含收敛子列。（广义：有界数列必包含广义收敛子列）

定理1.3.4 （Heine-Borel有限覆盖定理）

有限闭区间的任意开覆盖中，必定存在有限个子覆盖。

（Cauchy收敛准则，单调有界定理在前文）

{an}⊂R\{ a_{n} \} \subset \mathbf{R}{an​}⊂R ，若 ∀A>0\forall A>0∀A>0 ，∃N\exists N∃N 使得 ∀n>N\forall n>N∀n>N 都有 an>Aa_{n}>Aan​>A ，则称 an→+∞a_{n}\to +\inftyan​→+∞ ，有广义极限 +∞+\infty+∞ 。

定义1.4.2 （无穷小数列的定义）

若 an→0a_{n}\to 0an​→0 ，则称 {an}\{ a_{n} \}{an​} 是无穷小数列，简称无穷小。

定义1.4.3 （无穷大数列的定义）

若 an→∞a_{n}\to \inftyan​→∞ ，则称 {an}\{ a_{n} \}{an​} 是无穷大数列，简称无穷大。

性质1.4.5 （无穷大/无穷小的性质）

（1）无穷小（大）的和（差/积）为无穷小（大）
（2）{an}\{ a_{n} \}{an​} 无穷小（大）， {cn}\{ c_{n} \}{cn​} 有界， {ancn}\{ a_{n}c_{n} \}{an​cn​} 为无穷小（大）
（3）若对于充分大的 nnn ，有 0≤∣an∣≤∣bn∣0\le |a_{n}|\le |b_{n}|0≤∣an​∣≤∣bn​∣ 。若 {bn}\{ b_{n} \}{bn​} 无穷小，则 {an}\{ a_{n} \}{an​} 为无穷小；若 {an}\{ a_{n} \}{an​} 无穷大，则 {bn}\{ b_{n} \}{bn​} 无穷大。
（4）无穷大数列必无界；无穷大数列任意子列必为无穷大数列；无界数列必有无穷大子列。
