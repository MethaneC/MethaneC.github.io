Part 2 函数极限与连续性

定义2.1.1 （blablabla）

映射、函数、定义域、值域、像、原像、满射、单射、双射、逆映射、映射复合、恒等映射、复合函数、单调函数、反函数、初等函数、奇函数、偶函数、周期函数

定义2.1.2 （逆像）

设映射 \(f:A\to B\) 并且 \(F \subset B\) 。则 \(A\) 定义 \[ f^{-1}(F):=\{ x\in A|f(x)\in F \} \] 为 \(F\) 的逆像。

性质2.1.3 （反函数性质）

在定义域上严格递增（递减）的函数一定存在反函数，并且反函数也严格递增（递减）。

定义 2.2.1 （函数极限）

若 \(f(x)\) 在 \(x_{0}\) 的空心邻域中有定义。 \(l\in \mathbf{R}\) ，若 \(\forall \epsilon>0\) ，\(\exists \delta>0\) 使得 \(0<|x-x_{0}|<\delta\) 的 \(\forall x\) 都有 \[ |f(x)-l|<\epsilon \] 则称 \(x\) 趋于 \(x_{0}\) 时 \(f\) 趋于极限 \(l\) ，记为 \[ \lim_{ x \to x_{0} } f(x)=l,f(x)\to l(x\to x_{0}) \] 若存在极限，则 \(f\) 在 \(x_{0}\) 收敛，反之发散。

定义 2.2.2 （函数不收敛的定义/2.2.1的否定）

若 \(f(x)\) 在 \(x_{0}\) 的空心邻域中有定义。 \(l\in \mathbf{R}\) ，若 \(\exists \epsilon_{0}>0\) ，使得 \(\forall \delta>0\) ，存在 \(0<|x_{\delta}-x_{0}|<\delta\) 的 \(x_{\delta}\) 满足 \[ |f(x_{\delta})-l|\ge \epsilon_{0} \] 则称当 \(x\) 趋于 \(x_{0}\) 时 \(f\) 不收敛于 \(l\) ， \(f(x)\nrightarrow l(x\to x_{0})\) 。

性质 2.2.3 （函数收敛与数列收敛/Heine原理）

\(f(x)\to l(x\to x_{0})\) 的充要条件是，任意极限为 \(l\) 的数列 \(\{ x_{n} \}\) （ \(x_{n}\ne x_{0}\) ），都有 \(f(x_{n})\to l(n\to +\infty)\) 。

性质 2.2.4 （2.2.3的否定）

\(f(x)\nrightarrow l(x\to x_{0})\) 的充要条件是，存在 \(\{ x_{n} \}\) 满足 \(x_{n}\to x_{0}\) ，但是 \(f(x_{n})\nrightarrow l\) 。

性质 2.2.5 （函数极限唯一性）

若函数收敛，则任意两个收敛于 \(x_{0}\) 的数列，像数列一定收敛于同一个值。

性质2.2.6 （极限的四则运算）

若 \(f(x),g(x)\) 在 \(x_{0}\) 收敛，且 \(f(x)\to a,g(x)\to b\) ，那么： \[ \begin{aligned} &\lim_{ x \to x_{0} } (f \pm g)(x)=a\pm b \\ &\lim_{ x \to x_{0} } (fg)(x)=ab \\ &\lim_{ x \to x_{0} } \frac{f}{g}(x)=\frac{a}{b},(b \ne 0) \end{aligned} \]

性质2.2.7 （极限的复合）

若 \(f(x)\to l(x\to x_{0})\) ，\(g(t)\to x_{0}(t\to t_{0})\) ，并且在 \(t_{0}\) 邻域内 \(g(t)\ne x_{0}\) ，则 \[ \lim_{ t \to t_{0} } (f \circ g)(t)=l \]

性质2.2.8 （Cauchy收敛准则）

\(f\) 在 \(x_{0}\) 存在极限的充要条件是，对于 \(\forall \epsilon>0\) ， \(\exists \delta>0\) 使得 \(0<|x_{1}-x_{0}|<\delta,0<|x_{2}-x_{0}|<\delta\) 的 \(\forall x_{1},x_{2},\) ，都有 \[ |f(x_{1})-f(x_{2})|<\epsilon \]

性质2.2.9 （极限的不等式性质）

若 \(f,g\) 在 \(x_{0}\) 均有极限，且存在邻域使得 \(f(x)\le g(x)\) ，则 \[ \lim_{ x \to x_{0} } f(x)\le \lim_{ x \to x_{0} } g(x) \]

性质2.2.10 （夹逼定理）

\(f,g,h\) 在 \(x_{0}\) 处有极限，且存在邻域使得 \(f(x)\le g(x)\le h(x)\) ，极限值 \[ \lim_{ x \to x_{0} } f(x)=\lim_{ x \to x_{0} } h(x)=l \] 则 \[ \lim_{ x \to x_{0} } g(x) =l \]

性质2.2.11 （两个重要极限）

\[ \begin{aligned} &\lim_{ x \to 0 } \frac{\sin x}{x}=1 \\ &\lim_{ x \to \infty } \left( 1+\frac{1}{x} \right)^{x}=e \end{aligned} \]

性质 2.2.12 （收敛必有界）

若函数在 \(x_{0}\) 收敛，则在 \(x_{0}\) 邻域内有界

定义2.2.13 （单边极限）

设 \(l\in \mathbf{R}\) ，若 \(\forall \epsilon>0\) ，\(\exists \delta>0\) ，使得 \(x_{0}-\delta<x<x_{0}\) 的 \(\forall x,\) 有 \[ |f(x)-l|<\epsilon \] 那么 \(l\) 称为 \(f\) 在 \(x_{0}\) 处的左极限，记为 \[ l=\lim_{ x \to x_{0}^{-} } f(x)=f(x_{0}^{-}) \]

设 \(l\in \mathbf{R}\) ，若 \(\forall \epsilon>0\) ，\(\exists \delta>0\) ，使得 \(x_{0}<x<x_{0}+\delta\) 的 \(\forall x,\) 有 \[ |f(x)-l|<\epsilon \] 那么 \(l\) 称为 \(f\) 在 \(x_{0}\) 处的右极限，记为 \[ l=\lim_{ x \to x_{0}^{+} } f(x)=f(x_{0}^{+}) \]

同样可以证明单边极限的 2.2.2 - 2.2.11（条件需要修改）

性质2.2.14 （左右极限的性质）

函数 \(f\) 在 \(x_{0}\) 收敛的充要条件是左右极限相等，且值为左右极限。

定义2.2.15 （广义极限）

\(l\in \mathbf{R}\) ，若 \(\forall \epsilon>0\) ，\(\exists A>0\) ，使得所有 \(x>A\) 都有，\(|f(x)-l|<\epsilon\) ，则称 \(f(x)\to l,(x\to +\infty)\) 。

\(l\in \mathbf{R}\) ，若 \(\forall \epsilon>0\) ，\(\exists A<0\) ，使得所有 \(-x>A\) 都有，\(|f(x)-l|<\epsilon\) ，则称 \(f(x)\to l,(x\to -\infty)\) 。

\(l\in \mathbf{R}\) ，若 \(\forall \epsilon>0\) ，\(\exists A>0\) ，使得所有 \(|x|>A\) 都有，\(|f(x)-l|<\epsilon\) ，则称 \(f(x)\to l,(x\to \infty)\) 。

\(x_{0}\in \mathbf{R}\) ，若 \(\forall A>0\) ，\(\exists \delta>0\) ，使得 \(\forall x, 0<|x-x_{0}|<\delta\) ，都有 \(f(x)>A\) ，则 \(f(x)\to +\infty(x\to x_{0})\)

\(x_{0}\in \mathbf{R}\) ，若 \(\forall A>0\) ，\(\exists \delta>0\) ，使得 \(\forall x, 0<|x-x_{0}|<\delta\) ，都有 \(-f(x)>A\) ，则 \(f(x)\to -\infty(x\to x_{0})\)

\(x_{0}\in \mathbf{R}\) ，若 \(\forall A>0\) ，\(\exists \delta>0\) ，使得 \(\forall x, 0<|x-x_{0}|<\delta\) ，都有 \(|f(x)|>A\) ，则 \(f(x)\to \infty(x\to x_{0})\)

\(x_{0}\) 为无穷时同样可以定义收敛于无穷。

定义2.3.1 （无穷小的阶）

若 \(f,g\) 在 \(x_{0}\) 处的极限均为 \(0\) ，并且 \(g\) 在 \(x_{0}\) 空心邻域不为 \(0\)

（1） \(\frac{f(x)}{g(x)}\to 0\) ，则称 \(f\) 是 \(g\) 的高阶无穷小

（2） \(\frac{f(x)}{g(x)}\to l\in \mathbf{R}\) ，则称 \(f\) 与 \(g\) 是同阶无穷小

（3） \(\frac{f(x)}{(x-x_{0})^{\alpha}}\to l\) ，则称 \(f\) 是 \(\alpha\) 阶的无穷小

（4） \(\frac{f(x)}{g(x)}\to 1\) ，则称 \(f\) 与 \(g\) 是等价无穷小

定义2.3.2 （无穷大的阶）

若 \(f,g\) 在 \(x_{0}\) 处的极限均为 \(\infty\) ，并且 \(g\) 在 \(x_{0}\) 空心邻域不为 \(0\)

（1） \(\frac{f(x)}{g(x)}\to 0\) ，则称 \(g\) 是 \(f\) 的高阶无穷大

（2） \(\frac{f(x)}{g(x)}\to l\in \mathbf{R}\) ，则称 \(f\) 与 \(g\) 是同阶无穷大

（3） \(\frac{f(x)}{g(x)}\to 1\) ，则称 \(f\) 与 \(g\) 是等价无穷大

定义2.3.3 （O o记号）

（1）当 \(x\to x_{0}\) 时， \(\frac{f(x)}{g(x)}\) 有界，则 \[ f(x)=O(g(x)) \] （2）当 \(x\to x_{0}\) 时，\(\frac{f(x)}{g(x)}\to 0\) ，则 \[ f(x)=o(g(x)) \] （3）特别地， \(f(x)=O(1)\) 表示 \(f\) 有界， \(f(x)=o(1)\) 表示 \(f\) 是无穷小。

性质2.3.4 （O o记号）

\[ \begin{aligned} &o(u)+o(v)=o(u+v)\\ &o(u)o(v)=o(uv) \\ &uo(v)=o(uv) \end{aligned} \]

定义2.4.1 （函数连续）

若 \(f\) 在 \(x_{0}\) 的邻域上有定义，且 \[ \lim_{ x \to x_{0} } f(x)=f(x_{0}) \] 则称 \(f\) 在 \(x_{0}\) 处连续。

与函数左右极限一样，可定义左连续右连续。函数连续的充要条件是左右连续。

若 \(f\) 在开区间 \((a,b)\) 上连续，当且仅当 \(\forall x_{0}\in(a,b)\) ， \(f\) 在 \(x_{0}\) 连续。

若 \(f\) 在闭区间 \([a,b]\) 上连续，当且仅当 \(\forall x_{0}\in(a,b)\) ， \(f\) 在 \(x_{0}\) 连续，且 \(a\) 处右连续， \(b\) 处左连续。 同样可以推广到端点为无穷。

若 \(f\) 在 \(I\) 上连续，记 \(f\in C(I)\) ，如 \(f\in C(R),f\in C([a,b])\)

定义2.4.2 （函数连续的性质）

（1）若 \(f,g\) 在 \(x_{0}\) 连续，则 \(f\pm g,f\cdot g\) 在 \(x_{0}\) 连续。

（2）若 \(f,g\) 在 \(x_{0}\) 连续， \(g(x_{0})\ne 0\) ，则 \(\frac{f}{g}\) 在 \(x_{0}\) 连续。

（3）若 \(g\) 在 \(t_{0}\) 连续， \(f\) 在 \(g(t_{0})\) 连续 ，则 \(f \circ g\) 在 \(t_{0}\) 连续。

（4）若 \(f\) 在 \(x_{0}\) 连续，且在邻域上单调，则 \(f^{-1}(x)\) 在 \(f(x)\) 连续。

（5）所有初等函数都在其定义域上连续

（这些性质可以用来证明区间连续，一致连续）

定义2.4.3 （间断点）

（1）第一类间断点（两端均存在极限但不连续）

  1. 跳跃间断点 \(f(x^{+}) \ne f(x^{-})\)

  2. 可去间断点 \(f(x^{+}) = f(x^{-})\)

（2）第二类间断点（至少有一个无限或不存在）

  1. 无穷间断点 \(f(x^{+}),f(x^{-})\) 至少有一个为 \(\infty\)

  2. 震荡间断点 \(f(x^{+}),f(x^{-})\) 至少有一个不存在

定义2.4.4 （振幅）

定义 \(f\) 在 \(x_{0}\) 点的振幅为 \[ w_{f}(x_{0}):=\lim_{ \delta \to 0^{+} } (\sup_{0<|x-x_{0}|<\delta} f(x)-\inf_{0<|x-x_{0}|<\delta} f(x)) \] 可以证明： \(f\) 在 \(x_{0}\) 连续的充要条件是 \(w_{f}(x_{0})=0\)

定义2.4.5 （一致连续）

若 \(\forall\epsilon>0\) ， \(\exists \delta>0\) ，使得对于满足 \(|x_{1}-x_{2}|<\delta\) 的 \(\forall x_{1},x_{2}\in I\) ，有 \[ |f(x_{1})-f(x_{2})|<\epsilon \] 则称 \(f(x)\) 在 \(I\) 上一致连续。

定义2.4.6 （2.4.5的否定）

若 \(\exists\epsilon_{0} >0\) ， \(\forall \delta>0\) ， \(\exists x_{\delta},x_{\delta}'\in I\) ，满足 \(|x_{\delta}-x_{\delta}'|<\delta\) 但是 \[ |f(x_{\delta})-f(x_{\delta}')|\ge \epsilon_{0} \] 则称 \(f(x)\) 在 \(I\) 上不一致连续。

等价定义： \(\exists\epsilon_{0} >0\) ， \(\forall n\in \mathbf{N^{*}}\) ， \(\exists s_{n},t_{n}\in I\) ，满足 \(|s_{n}-t_{n}|< \frac{1}{n}\) 但是 \[ |f(s_{n})-f(t_{n})|\ge \epsilon_{0} \]

（事实上， \(\frac{1}{n}\) 可以替换为任意趋向于 \(0\) 的数列）

性质2.4.7 （一致连续的性质）

（1）有限闭区间连续函数必一致连续

若 \([a,b]\) 是有限闭区间， \(f\in C([a,b])\) ，则 \(f\) 在 \([a, b]\) 一致连续

（2）开区间一致连续判别法

若 \(f\) 在 \((a,b)\) 一致连续的充要条件是 \(f\) 在 \((a,b)\) 连续且 \(f(a^{+}),f(b^{-})\) 存在且有限

定义2.4.8 （连续模）

定义 \(f\) 在 \(I\) 上的连续模为 \[ w_{f}(I):=\lim_{ \delta \to 0^{+} } \sup_{|x_{1}-x_{2}|<\delta,x_{1},x_{2}\in I} |f(x_{1})-f(x_{2})| \] 可以证明 \(f\) 在 \(I\) 上一致连续的充要条件是 \(w_{f}(I)=0\)

性质2.5.1 （有限闭区间连续函数必有界）

若 \(f\in C([a,b])\) ，则 \(f\) 在 \([a,b]\) 上有界

性质2.5.2 （有限闭区间连续函数必能取到确界）

若 \(f\in C([a,b])\) ，则 \(\exists x^{*},x_{*}\in[a,b]\) 使得 \[ f(x^{*})=\sup_{x\in[a,b]} f(x), f(x_{*})=\inf_{x\in[a,b]} f(x) \]

性质2.5.3 （零值定理）

\(f\in C[a,b]\) ，并且 \(f(a)f(b)<0\) ，则 \(\exists x_{0}\in [a,b]\) 使得 \(f(x_{0})=0\)

性质2.5.4 （介值定理）

若 \(f\in C([a,b])\) ，且 \(f\) 不是常函数，则 \(f([a,b])\) 是一个有限闭区间，即 \[ f([a,b])=[\inf_{x\in[a,b]} f(x), \sup_{x\in[a,b]} f(x)] \]

定义2.6.1 （函数的极限点集合 \(E\)）

\[ E:=\{ l\in R_{\infty}|\exists x_{n} \subset I,x_{n}\to l,\text{s.t.} f(x_{n})\to l \} \]

性质2.6.2 （ \(E\) 的性质）

\(E\) 非空。 \(\sup E \in E,\inf E\in E\) 。

任何 \(\{ x_{n} \}\) 满足 \(x_{n}\to x_{0}\) ，\(\{ f(x_{n}) \}\) 极限点集合记为 \(E_{0}\) ，则 \(E_{0}\subset E\) 。

定义 2.6.3 （上下极限定义一）

\[ \begin{aligned} \limsup_{ x \to x_{0}} f(x) &:= \sup E \\ \liminf_{ x \to x_{0}} f(x) &:= \inf E \end{aligned} \]

定义 1.5.4 （上下极限定义二）

\[ \begin{aligned} \limsup_{ x \to x_{0} } f(x) &:=\lim_{ \delta \to 0 } \sup_{0<|x-x_{0}|<\delta} f(x) \\ &= \inf_{ \delta \to 0 } \sup_{0<|x-x_{0}|<\delta} f(x) \\ \liminf_{ x \to x_{0} } f(x) &:=\lim_{ \delta \to 0 } \inf_{0<|x-x_{0}|<\delta} f(x) \\ &= \sup_{ \delta \to 0 } \inf_{0<|x-x_{0}|<\delta} f(x) \\ \end{aligned} \]

定义 1.5.5 （上下极限定义三）

若 \(l\) 满足： （1）对于 \(\forall \epsilon>0\) ， \(\exists \delta>0\) ，使得满足 \(0<|x-x_{0}|<\delta\) 的 \(\forall x\) 有 \[ f(x)<l+\epsilon \] （2）对于 \(\forall \epsilon>0\) ，\(\exists \{ x_{n} \}\) ，\(x_{n}\to x_{0}\) 使得 \[ f(x_{n}) > l-\epsilon \] 则 \[ \limsup_{ n \to \infty } f(x) := l \]

性质1.5.6 （上下极限与极限）

\[ \liminf_{ x \to x_{0} } f(x) \le \limsup_{ x \to x_{0} } f(x) \] 取等条件条件：当且仅当， \(f(x_{0})\) 收敛，其极限等于其上下极限的值。

性质1.5.7 （上下极限的不等式性质）

若对 \(x_{0}\) 某个邻域中 \[ f(x)<g(x) \] 则 \[ \begin{aligned} &\limsup_{ x \to x_{0} } f(x) \le \limsup_{ x \to x_{0} } g(x) \\ &\liminf_{ x \to x_{0} } f(x) \le \liminf_{ x \to x_{0} } g(x) \end{aligned} \] （同样具有模糊保号性）
